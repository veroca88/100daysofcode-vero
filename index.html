CHEAT SHEET

.map() method : creates a new array from calling a specific function on each item in the principal array. This method doesn't filter or extract if the condition is true, this method interact or change with all the elements of an array. If we are working with an array map method will apply for each index, and if we are workin with a nested array it will apply to the macro index. - Calling a function on each item in an Array ? if we have an initial array const oddNumber = [1, 3, 5, 7, 9] we can use a callback function (callback oddNumber) inside map method, we can modify each individual item in an array and create a new array off it. (Always .map will work with all elements of an array).

const oddNumber = [1, 3, 5, 7, 9] const dobleNumber = oddNumber.map(number => { return number * 2 } console.log(dobleNumber)

Or we can be more cleaner with:

const makeDobleNUmber = dobleOddNumber => dobleOddNumber * 2; // here we stablished a variable with the parameter of the function const dobleNumber = oddNumber.map(makeDobleNUmber); console.log(dobleNumber)

We can use .call() method with string to change to an array, we have to remember that if we use call method with map method, map apply in all (letters of the word), everything in javascript is an object, .call() allows us to utilize the context of one object on another, also can be passed arguments of the context to be used, and "parameters for the arguments of the original function"

//Using map() with call() const fruit = ['apple', 2] const map = Array.prototype.map const fruitA = map.call(fruit, letter => { return `${letter}a` }) console.log(fruitA) //output ["applea", "2a"] //Using only map() const fruit = ['apple', 2] const fruitA = fruit.map(letter => { return `${letter}a` }) console.log(fruitA) //output ["applea", "2a"]

In the same way that we can iterate and modify inside an array, we can reformatting array object, we can modify our object inside of an array and return a new one

const myFavoriteSongs = [ { artist: 'Felix Jaenh', song: 'Book of love', timesPlay: 4}, { artist: 'The temper trap', song: 'Sweet disposition', timesPlay: 3} ] const mostPopularSong = myFavoriteSongs.map(bestSong => { const container = {}; container[bestSong.artist] = bestSong.song; container.timesPlay = bestSong.timesPlay * 5; return container; }) console.log(mostPopularSong)

This use case can be employed to process or condense received data before being saved or parsed on a frontend application.
